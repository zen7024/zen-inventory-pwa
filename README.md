# zen-inventory-pwa

スマホのカメラでバーコード・QRコードをスキャンして
在庫を管理できるツールです。
倉庫現場20年の経験をもとに、「現場で本当に使える」ことを優先して開発しました。

## できること
- バーコード・QRコードをカメラでスキャン
- 保管場所（棚番号など）を記録
- スキャン履歴をリアルタイムで表示
- CSVファイルでデータをダウンロード

## 使い方
1. `index.html` をブラウザで開く
2. カメラへのアクセスを許可する
3. バーコードにカメラを向けてスキャン
4. 保管場所を入力する（例: A01-01）
5. 「CSVダウンロード」で記録を保存

## 対象デバイス
- スマホ対応（iPhoneのSafari・AndroidのChrome）
- PWA対応（ホーム画面に追加して使える）

## 使用技術
- HTML / JavaScript
- html5-qrcode（バーコード読み取りライブラリ）
